# 리엑트로 Todo List 만들기

## 사용법

### 1. 자신의 깃에 fork 하기

  ![캡처](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/0aced699-bc84-4ae5-a3ad-5b745c753802)
  
### 2. Git Acrion 설정 바꾸기

  #### Fork한 레포지토리 -> Settings -> Actions -> General
  
  ![f](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/f684428a-d55b-4421-9c02-3d090df00570)

  #### Workflow permissions에 Read and write Permissions 체크 후 Save
  
  ![ddd](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/09cd99fc-5c80-47fe-9ede-445d98df236e)

### 3. package.json 변경

  #### homepage를 ```https://(자신의깃아이디).github.io/todo-list``` 로 변경 후 커밋 & 푸시
  
  ![ff](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/a2e337bf-9094-49dd-8167-fecbd268d315)

### 4. Action이 완료되면 gh-pages 브렌치 생성

### 5. Pages 설정

  #### Settings -> Pages

  ![sAD](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/eaf018f5-66ba-479d-844d-3eeb6a68c8c2)

  #### Source를 Deploy from a branch 로 설정

  #### Branch를 gh-pages /(root) 로 설정
  
  ![ASDFAFD](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/89cf7174-0199-4d0e-b13b-7a1a82cada1e)

  #### Action이 완료되면 배포 완료 -> [https://(자신의깃아이디).github.io/todo-list](https://Untoc-Web-Develop.github.io/todo-list) 로 접속
  
  ![DASD](https://github.com/Untoc-Web-Develop/todo-list/assets/51306225/ebd4ccb9-4659-4392-a0ae-60f4cbe9b97d)
